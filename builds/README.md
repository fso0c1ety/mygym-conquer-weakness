# APK Builds

This directory contains automatically generated APK builds from GitHub Actions.

## Latest Builds

APK files are automatically built and committed when changes are pushed to the `main` branch.

Each build is named with a timestamp: `mygym-YYYYMMDD-HHMMSS.apk`

## Download

To install on your Android device:
1. Download the latest APK file
2. Enable "Install from Unknown Sources" in your device settings
3. Open the APK file to install

## Manual Build

You can also trigger a manual build by going to:
- Actions → Android Debug APK (Capacitor) → Run workflow
